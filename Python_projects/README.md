# 🐍 Python Projects Repository

This repository contains Python-based projects created as part of my learning journey from the **100 Days of Code: Python Bootcamp (Udemy)**.

The projects range from beginner to intermediate level and are built to strengthen my programming skills and problem-solving ability. New projects are added daily as I progress through the course.

## 📌 Projects Included

### 🎸 Day 1 - Band Name Generator

A simple Python project that generates a band name based on user input.

**Concepts used:**

- Variables
- Input/Output
- String concatenation

**Example:**

```
City name: Delhi
Pet name: Bruno
Band Name: Delhi Bruno
```

---

### 💰 Day 2 - Tip Calculator

A Python program to calculate how much each person should pay after splitting the bill, including tip.

**Concepts used:**

- Data types
- Type casting
- Mathematical operations

**Example:**

```
Total bill: 1000
Tip: 10%
People: 2
Each person pays: 550
```
